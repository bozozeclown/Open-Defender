System Prerequisites

Before proceeding with the installation, ensure your system meets the following requirements.
Hardware Requirements
  • CPU: 4 cores minimum (8 cores recommended)
  • RAM: 8GB minimum (16GB recommended)
  • Storage: 50GB free space (SSD recommended)
  • Network: Stable internet connection for downloading dependencies
Software Requirements
  • Docker: Version 20.10 or higher
  • Docker Compose: Version 1.29 or higher
  • Kubernetes: Version 1.22+ (for K8s deployment)
  • kubectl: Configured to connect to your cluster
  • Helm: Version 3.5+ (for Helm deployment)
External Dependencies
PostgreSQL Database

Version 12 or higher. Will be installed in the next step.

Redis Cache

Version 6 or higher. Used for session management and caching.

Create Database

Follow these steps to create the database and user for your security monitoring system:

Start PostgreSQL Service

Ensure the PostgreSQL service is running on your system.

sudo systemctl start postgresql
sudo systemctl enable postgresql
Connect to PostgreSQL

Connect to PostgreSQL as the default postgres user.

sudo -u postgres psql
Create Database and User

Run the following SQL commands to create the database and user:

CREATE DATABASE security_monitoring;
CREATE USER security_user WITH PASSWORD 'your_secure_password';
GRANT ALL PRIVILEGES ON DATABASE security_monitoring TO security_user;
\q
Verify Database Creation

Verify that the database was created successfully:

sudo -u postgres psql -c "\l"
